URL Shortener Microservice

Documentation

Welcome to the URL Shortener service. This documentation should supply you with all the information you need to start making requests. It is a really simple api service built to complete one of the projects for the freeCodeCamp Back End Development and APIs certification.

Base Url
Create Short Url Endpoint

An endpoint that will accept a valid url and create a short url for it and return that short url as a JSON object.

/api/shorturl
Example Request:
POST

Content-Type:
application/x-www-form-urlencoded

url:
The url to be shortened
Example Response:
{
 "original_url": "https://freecodecamp.org",
 "short_url": "5HnSOll6pcVs",
 "_id": "6275425ad2fe0db53499fb59",
 "__v": 0,
}
Try It Out
Get Short Url Endpoint

An endpoint that will accept a short url and redirect you to the corresponding original url.

/api/shorturl/{url}
Example Request:

GET
